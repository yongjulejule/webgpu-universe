# Conway's life game using WebGPU

[conway](conway.mov)

# Run 

## requirement 

- google chrome
- pnpm (or npm or yarn)

```bash
pnpm install
pnpm build
# (sleep 3; open -a "Google Chrome" http://localhost:3000/gpu ) &
pnpm start 
```
